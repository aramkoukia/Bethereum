Solidity Distributed BlockChain Application

Install bloc

Download from npm

`npm install blockapps-bloc`

Initialize a project

bloc init

Install npm dependencies

cd app_name && npm install

Generate a key

bloc genkey

You will see an out put something like 

app/users/admin/6f29e2b2b6b922800fd959a94b88cd305790413a.json

Remember the username and password, you will need to enter this in visual studio.

Run Your local bloc keyserver

bloc start

bloc should now be listening on localhost:8000

Create a distributed application in Visual Studio

  • Click new project
  • From the template select `Solidity` & DecentralizedApplication
  • Navigate to the Solution Explorer and right click on your project
  • Select Properties
  • Enter the password you used with the `bloc genkey` command in the BlockchainPassword field
  • Click apply

Compile and deploy your first smart contract

  • In Visual Studio in `samples` right click one of the `.sol` files and click `compile`
  • Now right click and click `deploy`
  • You should see your browser open with generated html page based on the methods in the solidity contract
  • If you do not look in the visual studio console and you will see a link to the deployed contract
  • Click this link and you will see and generated html page based on the methods in the contract
